# FC Contact Page

A responsive **Contact Us** page built with plain HTML, CSS, and JavaScript.

## Features

- Two-panel card layout (info panel + form panel)
- Fields: Name, Email, Phone, Product (dropdown), Message
- Client-side form validation with inline error messages
- Success banner on valid submission
- Fully responsive — stacks on mobile
- Font Awesome icons for contact details
- Inter font via Google Fonts

## Usage

Open contact.html directly in a browser — no build step needed.  
Replace the submit handler stub in contact.js with your backend / form service call.
